# AgroFuturo 2026 - Plataforma Web Premium

## 📋 Tema da Página
**"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"**
A interface explora conceitualmente a convergência entre o vigor produtivo do agronegócio tradicional brasileiro e as soluções computacionais e biotecnológicas futuristas (como Inteligência Artificial, Drones e Microbiologia).

## 🎯 Objetivo da Página
Servir como um hub de autoridade informativa e captação (Landing Page estratégica) para demonstrar como a inovação digital e biológica reposiciona o Brasil como líder em sustentabilidade no campo, culminando na conversão de leads interessados por meio de um seminário online focado em IA aplicada à lavoura.

## 🛠️ Instruções de Uso e Implantação

### 1. Organização dos Arquivos
Certifique-se de salvar os três códigos fornecidos exatamente com os seguintes nomes na mesma pasta raiz do projeto:
- `index.html` (Estrutura estrutural e conteúdo semântico)
- `style.css` (Folha de estilo responsiva e tokens visuais)
- `script.js` (Lógica de interatividade e acessibilidade nativa)

### 2. Inserção das Imagens Exigidas
O layout foi rigorosamente preparado com espaços reservados de visual premium que respeitam a estrutura exigida. Para visualizar as imagens reais do seu projeto, salve os arquivos de imagem desejados na mesma pasta com os seguintes nomes correspondentes:
1. **`Foto1.png`**: Representará a biotecnologia/microbiologia no campo.
2. **`Foto2.png`**: Representará drones e coleta de dados aéreos multiespectrais.
3. **`Foto3.png`**: Representará a automação por meio de maquinários inteligentes e tratores guiados por IA.

### 3. Execução
Basta dar um duplo clique no arquivo `index.html` para executá-lo em qualquer navegador moderno. Para testar as capacidades responsivas, utilize o modo de desenvolvedor do navegador (`F12`) simulando telas de smartphones (`320px`) até monitores UltraWide (`2560px`).

### 4. Recursos Especiais de Acessibilidade incorporados
- **Leitura por Voz**: Ao clicar no ícone de acessibilidade flutuante no canto inferior esquerdo, utilize o botão de áudio. O sistema ignorará os menus e lerá apenas o corpo textual informativo de forma nativa utilizando a API SpeechSynthesis.
- **Modo Escuro**: Alterne dinamicamente o tema no botão de lua no painel.
